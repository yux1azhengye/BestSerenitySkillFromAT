# 单股报告模板

> 一份围着这只票的研报，结论先行，教学融在分析里。方法内化、隐形：报告围着标的本身展开，不围着 Serenity 展开。

---

## 一句话结论 (thesis)

它是什么 (哪类卡点/哪类公司) + 客观空间 (bear/base/bull 关键档、是否过翻倍门槛) + **方向性买卖定档** (由综合判定得出，始终给):

| 定档 | 触发条件参考 |
|------|------------|
| Fire Sale | 好卡点判据命中 >=5 条 + 估值低于 bear 档 + 近期催化剂明确 + 无硬否决红旗 |
| Extremely Strong Buy | 好卡点判据命中 >=4 条 + 估值处于 bear-base 之间 + 无硬否决红旗 |
| Buy | 好卡点判据命中 >=3 条 + 估值处于 base 档附近 + 红旗可管理 |
| Hold | 卡点真实但估值已充分反映 (base-bull 之间) + 或存在可管理的红旗 |
| Sell | 踩中硬否决红旗 (无限 ATM/零收入纯炒作) + 或 thesis 核心论点已破裂 |

> 大盘 (市值远超 Sub-$2B 且已被充分发现) 改为 "判断力弱、只给卡点定性不给档"。

一句因由。**这是用户的决策输入，先行，不复述推导。**

按需 (仅用户要他的 take/要据他下注时): 再补一句他本人的具体立场或与客观定档的分歧。

## 事实底座 (建认知，教学重点)

> 每个数配一句 "意味着什么" (中性，帮非专业读者读懂)。缺数标 `[未核实]`。

- 业务拆分/各占比
- 市值/股本
- 营收/毛利/净利
- backlog/订单
- 客户集中度与点名客户
- 现金/债务
- 产业链定位: 在哪一层、上下游是谁、这层为什么卡

---

## 分析: 是不是好卡点 + 该值多少

### 是不是卡点 (判断显形)

顺钱流定位后，**用大白话点明命中哪几条好卡点判据、踩哪条红旗**。不是只甩 "已 re-rate / 已定价" 的断言。尝试 >=1 条 OSINT/多跳推断 (受虚构守门约束)。

### 该值多少 (客观预期空间)

判类型 -> 套尺子 (超成长/紧缺卡点/深价值) -> 客观对标 + 显式加权 -> **bear/base/bull 三档，每档绑死假设** (对标谁/几倍/哪年)。含 `[推断]`/`[推测]` 假设的按精度降级只给数量级/方向。

---

## 结论与边界

- 值不值得的方向性结论 (不要 "各有优劣")
- **什么会证伪** (证伪门，即使没要也必须给)
- 适用边界 (大市值已定价/能力圈外直说判断力弱)
