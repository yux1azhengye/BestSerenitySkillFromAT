# 市场适配参考 (按需引用，非执行指令)

## A 股特色

### 差异对照

| 维度 | 美股原版 | A 股适配 |
|------|---------|--------|
| 政策权重 | 较低 (市场主导) | **极高** (政策直接催化板块) |
| 主力资金 | 对冲基金/机构 | **公募+私募+游资**，游资炒作凶猛 |
| 散户结构 | 较少 | **散户占比高**，情绪波动大 |
| 壳资源 | 无此概念 | 小市值有**壳价值**溢价 |
| 国产替代 | 非核心 | **核心逻辑之一** (卡脖子清单) |

### A 股独有信号维度

1. **龙虎榜**: 营业部游资席位暴露短线情绪
2. **融资融券**: 杠杆资金的态度 (融资余额暴涨 = 看多情绪)
3. **主力净流入**: 单日主力净流入 >5 亿 = 机构明显建仓
4. **政府工作报告**: 每年 3 月更新，重大方向性信号
5. **卡脖子清单**: 国务院发布，指引国产替代方向

### A 股工具箱

**neodata-financial-search** (行情/研报/资金):

```
"[板块名] 行情 资金流向"         # 板块整体行情
"[材料/元器件] 供需缺口 国产替代"  # 供应链关键环节
"[公司名] 券商研报 目标价"        # 研报信号
"[板块] 政策 支持"               # 政策催化
"[板块名] 成分股 A股"            # 板块成分股
```

**westock-data** (筹码/大宗/机构):

```
[代码].westock: chip_cost           # 筹码成本分布
[代码].westock: block_trade         # 大宗交易
[代码].westock: margin              # 融资融券余额
[代码].westock: institution_rating  # 机构评级
[代码].westock: hk_hold             # 北向资金持股
```

> **无工具降级**: 若环境未安装 neodata/westock MCP 工具，通过联网搜索巨潮资讯 (cninfo.com.cn)、东方财富 (eastmoney.com)、同花顺 (10jqka.com) 获取类似信息，标注数据来源和日期。

### A 股取数纪律 (与美股取数纪律平行)

美股一手源为 SEC filings (10-K/10-Q/8-K)，A 股一手源体系不同，须平行执行以下规则:

| 数据项 | A 股一手源 | 查找路径 |
|--------|-----------|---------|
| 营收结构/分部占比 | 年报 (年度报告) | 巨潮资讯 → 搜索公司名 → 年度报告 → "主营业务分行业/产品" |
| 客户集中度 | 年报 + 半年报 | 年报 → "主要客户" 或 "前五名客户" |
| 业绩预告/快报 | 交易所公告 | 深交所/上交所官网 → 公告查询 → 业绩预告/快报 |
| 问询函回复 | 交易所公告 | 巨潮资讯 → 搜索 "问询函" → 查看回复 (常含客户/供应商细节) |
| 大股东/减持 | 交易所公告 | 巨潮资讯 → 搜索 "减持" "权益变动" |
| 研报/目标价 | 券商研报 | 东方财富研报中心、同花顺研报频道 (仅作旁证，不当一手源) |
| 龙虎榜/大宗交易 | 交易所每日披露 | 深交所/上交所 → 交易公开信息 / 大宗交易 |
| 融资融券 | 交易所每日披露 | 交易所官网 → 融资融券数据 |
| 北向资金持股 | 港交所/交易所 | 沪深港通持股数据 (eastmoney 或 wind) |

**时效要求**:

- 年报披露窗口: 每年 1-4 月 (年报)、7-8 月 (半年报)、10 月 (三季报)
- 业绩预告窗口: 1 月 (年报预告)、7 月 (半年报预告)、10 月 (三季报预告)
- 快变字段 (龙虎榜/融资融券/大宗交易): 使用最近 5 个交易日数据，标注 as-of
- 问询函回复: 专门搜一次 as-of 前 90 天内的新问询函 (类似美股的 60 天 8-K 规则)

**虚构守门 (A 股版)**: 先在巨潮资讯找年报/半年报原文，找到标 `已证实`；真搜过仍找不到才标 `[推断]`。互动易/e互动平台的董秘回复可作 `管理层声称` 级别，不升为 `已证实`。

## 美股特色

### SEC Filings 路径

| 文件 | 关键信息 |
|------|---------|
| **10-K/10-Q/8-K** | 营收结构、客户集中度、backlog、稀释风险 (ATM/converts/warrants) |
| **Earnings transcripts** | 产能打满、认证将结束、明年放量、客户点名 |
| **Investor presentations** | 路线图、TAM、市场份额 |
| **S-3/ATM risk** | 稀释风险警示 |
| **Insider transactions** | 内部人买卖 (高位清仓 = 红旗) |
| **DEF 14A (Proxy)** | 管理层薪酬、股权激励 (判断利益一致性) |
| **13F/13D/13G** | 机构持仓变动 (判断 "机构是否在建仓") |

### 流动性/稀释风险

| 指标 | 阈值 | 含义 |
|------|------|------|
| Float | <20% | 挤压风险 |
| Short interest | >20% | 挤压潜力 |
| ATM programs | 持续增发 | 无限稀释 = 硬否决 |
| Convertibles/Warrants | 大额到期 | 潜在稀释 |
| Shelf registrations | 随时可增发 | 供给压力 |

## 全球市场

### Hong Kong (港股)

- **披露源**: HKEX filings (hkexnews.hk)、annual/interim reports、placings、connected transactions
- **关键关注**: 大陆政策暴露、流动性、南向资金资格 (Southbound eligibility)
- **典型卡点公司**: 舜宇光学 (光学)、丘钛科技 (摄像头模组)、华虹半导体 (代工)
- **特殊风险**: 老千股风险 (供股/合股/更名频繁的小市值公司 = 硬否决)

### Taiwan (台股)

- **披露源**: 公开资讯观测站 (mops.twse.com.tw)、月营收公告、法说会
- **关键关注**: 台积电供应链、日月光封装、联发科设计
- **特色**: 月度营收数据可追踪，比季报更及时

### Japan (日股)

- **披露源**: EDINET (金融厅电子开示)、会社四季报、公司 IR
- **关键关注**: 半导体设备 (东京电子/迪恩士)、材料 (信越化学/SUMCO)、光学 (HOYA)

### Korea (韩股)

- **披露源**: DART (金融监督院)、公司 IR
- **关键关注**: 三星/SK 海力士供应链 (HBM/DRAM/NAND)

### Europe (欧股)

- **披露源**: 各交易所 filings、公司 IR、trade journals
- **关键关注**: ASML (光刻)、意法半导体 (SiC/GaN)、Infineon (功率半导体)
- **特色**: 欧洲小盘往往被美国机构忽视 (Serenity 的 "欧洲小盘优先" 启发式)

### 通用注意事项

- FX/地缘政治暴露: 每个市场都需评估汇率风险和对华出口管制影响
- 数据源优先级: 当地交易所官方披露 > 公司 IR > 当地财经媒体 > 国际聚合站
